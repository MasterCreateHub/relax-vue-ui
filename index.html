简洁至上

Relax Vue UI 的组件通过丰富的属性配置即可实现大部分功能，大量减少模板代码，提升开发和维护效率。

优雅继承

Relax Vue UI 保留了 element-ui 的属性配置和使用习惯，确保开发者能够无缝迁移并享受更流畅的开发体验。

随心使用

Relax Vue UI 支持全量安装、按需引入及单个组件定制，灵活满足不同项目需求，轻松实现个性化开发。
