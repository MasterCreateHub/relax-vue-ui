# Converter

用于展示某个字段或对某个字段的值进行特别的展示

## Converter Usage

## Converter API

### Attributes

| 参数      | 说明                     | 类型                               | 可选值 | 默认值 |
| --------- | ------------------------ | ---------------------------------- | ------ | ------ |
| value     | 要转换的字段或要转换的值 | `Number,String,Boolean,Array,Date` | -      | -      |
| source    | 源数据或转换源数据的方法 | `Array,Object,Function`            | -      | -      |
| container | 最终的渲染容器           | `String`                           | -      | `span` |
