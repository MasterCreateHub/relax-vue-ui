# 指令概览

| 组件名称                      | 说明                                       |
| ----------------------------- | ------------------------------------------ |
| [`v-measure`](./measure.md)   | 监听元素尺寸大小变化,并在变化时触发回调    |
| [`v-emerge`](./emerge.md)     | 监听元素是否出现在指定视口范围内并触发回调 |
| [`v-readonly`](./readonly.md) | 为元素添加只读遮罩                         |
| [`v-disabled`](./disabled.md) | 为元素添加禁用遮罩                         |
